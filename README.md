## 📖 Overview
Managing inventory, library systems, or retail products requires thousands of unique barcodes. Generating them manually is tedious and prone to human error. This utility automates the entire process.

By inputting a range of serial numbers, this Flask application instantly generates standardized barcodes and compiles them into a perfectly formatted, high-resolution PDF document that is immediately ready for label printers.

## ✨ Key Features
- 🚀 **Bulk Generation**: Generate hundreds of sequential barcodes in seconds.
- 📄 **PDF Compilation**: Uses `ReportLab` to stitch the generated barcode images into a paginated PDF document.
- 🎨 **Custom Formatting**: Automatically handles spacing and margins for standard label printer sheets.
- 🌐 **Web Interface**: Clean, simple Flask UI for entering serial ranges and downloading the result.
